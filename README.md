Task Manager Dashboard
A simple and clean full-stack web application to manage daily tasks.

Features
Add Tasks: Quickly add new tasks to your list.

Delete Tasks: Remove completed or unwanted tasks.

Dashboard View: Get a quick overview of total tasks.

Analytics: Visual report using Chart.js to track task progress.

Tech Stack
Frontend: HTML5, CSS3, JavaScript

Backend: Python (Flask)

Database: SQLite

How to Run
Install Flask: pip install flask flask-cors

Run the backend: python backend/app.py

Open frontend/index.html in your browser.
